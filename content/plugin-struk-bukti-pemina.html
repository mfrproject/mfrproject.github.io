Plugin Struk Bukti Peminajaman SLiMS

Pemijaman buku pada umumnya tidak melampirkan bukti peminjaman, tetapi beberapa instansi menerapkan aturan tersendiri dalam peminjaman buku yaitu struk bukti peminjaman buku seperti yang diterapkan di Dinas Kearsipan dan Perpustakaan Kabupaten Indramayu. Plugin ini dibuat untuk menunjang kebutuhan di instansi tersebut. Ide untuk menciptakan plugin ini dicetuskan oleh salah satu staf bidang perpustakaan yaitu, Fathul Ilmi Nugraha.

Spesifikasi plugin ini minimal menggunakan perangkat pendukung seperti printer dot matrix seperti yang ada di wartel pada jaman itu lo :D dan kertas "tindes" atau apa lah itu namanya saya juga gak paham hehehe, Tapi ini tidak menutup kemungkinan untuk dapat digunakan pada printer selain yang sudah saya sebutkan diatas. Plugin ini pada dasarnya hanya mengantikan fungsi cetak struk yang sudah ada pada SLiMS. Pengaturan untuk plugin ini dapat diatur secara luwes melalui peraturan sistem pada menu cetak bukti sirkulasi.

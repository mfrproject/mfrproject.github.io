Plugin Ekspor Data Statistik Penggunaan Koleksi

Plugin ini merupakan plugin spesial dibulan ramadhan. Kegunaan dari plugin ini yaitu untuk mengekspor data statistik penggunaan koleksi yang ada di SLiMS. Plugin ini cukup sederhana, cara kerjanya pun mudah untuk dipahami. Prinsip nya plugin ini berkerja dengan mengambil seluruh data yang ada di tabel yang telah ditentukan lalu diekstrak dalam format excel maupaun csv, setelah proses selesai maka secara otomatis akan tercipta sebuah file yang siap di unduh dalam format .xls ataupun .csv. Plugin ini tidak saya buat dari nol melainkan saya racik dari beberapa skrip berbasis JavaScript yang saya unduh dari www.jqueryscript.net, lebih tepat nya pada laman ini.  

Bagaimana cara pemasangannya? anda dapat membacanya di laman github saya pada pranala ini. Lalu bagaimana mengunduhnya?
